## Что это

«Ответка» — маленькое десктопное приложение для Windows, которое помогает быстро отвечать в мессенджерах. Не можешь подобрать слова, лень печатать, хочется ответить красиво или с юмором — выделяешь сообщение собеседника, жмёшь горячую клавишу, и приложение предлагает **3 готовых варианта** в выбранном стиле.

Главное отличие от всех аналогов: **искусственный интеллект работает прямо на твоём компьютере.** Ничего не отправляется на чужие серверы, переписка остаётся только у тебя.

## Как пользоваться

1. **Выдели** сообщение собеседника и скопируй (или просто нажми горячую клавишу — приложение скопирует само).
2. **Нажми** `Ctrl + Shift + R` (клавишу можно поменять).
3. **Выбери** из трёх вариантов и кликни — ответ скопируется.
4. **Вставь** в чат: `Ctrl + V`.

## Возможности

- 🎭 **Стили ответа**: дружелюбно, формально, с юмором, коротко, флирт, вежливый отказ — и свои собственные.
- ✍️ **«Пиши как я»**: даёшь пару примеров своих сообщений — ответы получаются в твоей манере.
- ⚡ **Быстро**: ответ печатается на глазах, пара секунд на видеокарте.
- 💬 **Встроенный чат** с ИИ — полноценный локальный ассистент с историей.
- 🧠 **Выбор модели** под твоё железо — от лёгкой (для любого ноутбука) до максимальной.
- 🌗 Тёмная/светлая тема, русский и английский интерфейс, автозапуск.

## Установка

1. Скачай `Otvetka_*_x64-setup.exe` из раздела [**Releases**](../../releases).
2. Запусти установщик и следуй шагам.
3. При первом запуске выбери и скачай модель (один раз, ~1–2 ГБ) — приложение подскажет, какая подойдёт твоему компьютеру.
4. Готово. Значок живёт в трее у часов.

> При первом запуске Windows может показать синее окно «Windows защитила ваш компьютер» (так бывает у новых программ без платной подписи). Нажми **Подробнее → Выполнить в любом случае**.

## Почему это приватно

- Текст ответов придумывает нейросеть, **запущенная на твоём устройстве** ([llama.cpp](https://github.com/ggml-org/llama.cpp) + модель [Qwen](https://github.com/QwenLM/Qwen3)).
- Единственный выход в интернет — **разовое скачивание модели** при установке. Дальше приложение работает полностью офлайн.
- Никакой аналитики, аккаунтов и облака.

## Для разработчиков

<details>
<summary>Собрать из исходников</summary>

Нужны: [Rust](https://rustup.rs), Visual Studio Build Tools (C++), [Tauri CLI](https://tauri.app) (`cargo install tauri-cli`). Node.js не требуется — фронтенд на чистом HTML/JS.

```bash
# 1. Скачать движок llama.cpp в src-tauri/resources/llama:
powershell -ExecutionPolicy Bypass -File scripts/fetch-llama.ps1

# 2. Запуск в режиме разработки:
cargo tauri dev

# 3. Собрать установщик:
cargo tauri build
```

**Стек:** [Tauri 2](https://tauri.app) (Rust) — окна, трей, горячая клавиша, буфер обмена. [llama.cpp](https://github.com/ggml-org/llama.cpp) — движок инференса (Vulkan-сборка, работает на видеокартах NVIDIA/AMD/Intel, при отсутствии GPU — на процессоре). Модели: [Qwen3](https://huggingface.co/Qwen) в формате GGUF. Фронтенд — HTML/CSS/JS без сборщиков.

</details>
